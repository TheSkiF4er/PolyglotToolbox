# Contributing to Polyglot Toolbox

Thanks for wanting to help! Ways to contribute:
- Add an implementation in another language or improve existing ones.
- Improve docs, examples, or tests.
- Submit benchmarks or optimization PRs.

Rules:
- Small, focused PRs are best.
- Add README for your language describing build/run/test.
- Add at least one unit test.
- Follow code style used in language module.

Good first issues are labeled `good first issue`.
See `templates/module_template.md` to add a new language.
