This folder contains the PHP language implementation for the Polyglot Toolbox project.

## Overview
The PHP plugin is a simple web-based application demonstrating text processing and interoperability with the Polyglot Toolbox system.

## Files

- `public/index.php`: Main entry point for the PHP application.

## Installation

Ensure you have PHP installed.

```bash
# Start PHP built-in server from the project root
php -S localhost:8000 -t public
```

## Usage

```bash
# Open your browser and visit:
http://localhost:8000
```
