This folder contains the C++ language implementation for the Polyglot Toolbox project.

## Overview
The C++ plugin is a command-line tool that processes text input and outputs basic statistics, including:
- Number of characters
- Number of words
- Number of lines

## Files

- `textstats.cpp`: Main C++ source file for the text statistics tool.

## Build Instructions

Ensure you have a C++ compiler installed (e.g., g++).

```bash
# Compile the program
g++ -o textstats textstats.cpp
```

## Usage

```bash
# Run the program with input from a file
./textstats < input.txt
```

## Notes
- Designed to be simple and portable.
- Can be extended or integrated with other Polyglot Toolbox modules.
