This folder contains the Go language implementation for the Polyglot Toolbox project.

## Overview
The Go plugin is a command-line tool that demonstrates basic text processing and can interoperate with the Polyglot Toolbox core system.

## Files

- `main.go`: Main Go source file containing the application logic.

## Build Instructions

Ensure you have Go installed.

```bash
# Build the Go program
go build -o textstats main.go
```

## Usage

```bash
# Run the program with input from a file
./textstats < input.txt
```

## Notes
- Designed for simplicity and portability.
- Can be integrated with other language modules in Polyglot Toolbox.
