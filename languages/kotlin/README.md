This folder contains the Kotlin language implementation for the Polyglot Toolbox project.

## Overview
The Kotlin plugin is a console application that demonstrates basic text processing and can interoperate with other Polyglot Toolbox modules.

## Files

- `build.gradle.kts`: Gradle build script for the Kotlin project.
- `README.md`: This documentation file.

## Build Instructions

Ensure you have Kotlin and Gradle installed.

```bash
# Build the project using Gradle
./gradlew build
```

## Usage

```bash
# Run the Kotlin application
./gradlew run
```

## Notes
- Compatible with Kotlin 1.5+
- Can be integrated with other language modules in Polyglot Toolbox.
